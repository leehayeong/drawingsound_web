# Drawing Sound (web)
작곡 애플리케이션을 통해 악보를 생성하고, 생성한 악보의 **뷰어**와 **출력** 기능을 제공하는 웹 서비스
</br>
  * Front-end: BootStrap, Pug, Javascript, SCSS
  * Back-end: Express, Firebase
</br></br>
